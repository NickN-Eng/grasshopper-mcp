# Docs Guide

This folder collects architecture, MCP tooling, and testing references. Use this index to find the right document quickly.

## Suggested paths
- New developer: `dev-quickstart.md` -> `architecture.md` -> `tool-capabilities.md` -> `mcp.md`
- Testing/automation: `ai-testing-guide.md` -> `testing-architecture.md` -> `ai-testing.md`
- Planning/feature direction: `roadmap.md`

## Doc index
- `ai-testing-guide.md`: AI-focused workflow for automated testing, including scripts and log formats.
- `ai-testing.md`: Overview of AI testing goals and policies (shorter, policy-style doc).
- `architecture.md`: High-level system flow and key components.
- `bridge_mcp_csharp_suggestion.md`: Design suggestions for the C# bridge and MCP evolution.
- `dev-quickstart.md`: Build, deploy, and local dev workflow for both Python and C# pieces.
- `mcp.md`: Summary list of MCP tools and resources (compare with `tool-capabilities.md` for current behavior).
- `roadmap.md`: Planned features and sequencing.
- `testing-architecture.md`: Deep dive into the automated test harness and runtime environment.
- `tool-capabilities.md`: Current tool capabilities and known limitations (based on code).
